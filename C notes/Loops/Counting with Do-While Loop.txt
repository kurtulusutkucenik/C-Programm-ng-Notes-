#include <stdio.h>

int main() {

    int i, upper_limit;

    // Ask the user for the upper limit
    printf("Please enter the upper limit: ");
    scanf("%d", &upper_limit);

    i = 1; // Starting value
    do {
        printf("%d\n", i);
        i++; // Increment
    } while(i <= upper_limit); // Loop condition checked after execution

    return 0;
}

/* This program demonstrates counting from 1 to a user-specified number using a do-while loop:

Ask the user for an upper limit.

Initialize the counter i = 1.

The do block executes at least once, printing i and incrementing it.

After executing the block, the condition (i <= upper_limit) is checked.

The loop continues until the condition is false.

Key difference vs. while loop:

do-while executes at least once, even if the condition is false at the start. */
